# Digital Clock using HTML, CSS & JavaScript

A simple and responsive Digital Clock web application created using HTML, CSS, and JavaScript. The clock displays the current time in HH:MM:SS format with AM/PM indication and updates every second.

## Features

* Real-time digital clock
* AM/PM format
* Responsive design
* Hover animation effects
* Glowing UI design
* Built with beginner-friendly code

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```text
DigitalClock/
│
├── index.html
├── digital.css
└── digital.js
```

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in any web browser

## Output

The application displays:

```text
03:45:12 PM
```
